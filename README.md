# java-cli-maven-hsqldb-simple

## Description
A java maven build, that connects to hsqldb
database.

## Tech stack
- java
- maven
- hsqldb driver

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
